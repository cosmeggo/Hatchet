# Hatchet
A content-based Balatro mod that adds new Jokers, consumables, and more!

For a simplified Vanilla+ experience, try [Hatchet Lite](https://github.com/cosmeggo/Hatchet-Lite)

# Dependencies
- [Steamodded](https://github.com/Steamodded/smods) (Required)

This mod may not be compatible with other pre-existing mods. 

# Current Features
## Jokers
- 30/30 Common Jokers
- 30/30 Uncommon Jokers
- 30/30 Rare Jokers
- 10/30 Divine Jokers
- 1 Legendary Joker
## Consumables
- 10 Sephirot Cards
- 7 Divine Cards
## Modifiers
- 3 Enhancements
- 1 Edition
## Miscellaneous
- 5 Booster Packs
- 2 Stakes
- 4 Decks
- 4 Boss Blinds
- 1 Sticker
- 3 Challenges


# Planned Features

- Hatchet Lite
